Set up game playlists, resulting in automatically adding any videos of a certain game to a Youtube playlist. Essentially this allows users who stream multiple games to never worry about setting up custom playlists again. Define it here once, it will always happen.

This is completely optional. However it has been shown time and time again that grouping videos in playlists increases user retention.

To find your playlist ID navigate to your Youtube channel, go to Playlists, click the playlist you want. You should be redirected to a url that looks something like this: 'https://www.youtube.com/playlist?list=PL0_AOI...'. The playlist ID is part after 'list='.

Game names are case sensitive, please find all the specific game names here.
